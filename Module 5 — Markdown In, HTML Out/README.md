### Topic: Markdown In, HTML Out

Simple Summary

AI ko Markdown mein instructions do, aur AI se HTML mein output lo. Markdown AI ke liye easy hota hai, HTML insan ke liye readable aur shareable hota hai.

### Main Rule

Aap → AI

Markdown

Clear instructions / specification

AI → Aap

HTML

Beautiful page / report / dashboard

AI → AI

Markdown

Context / notes / specs

### Markdown Kya Hai?

Plain text + thori structure.

# Heading

Title ya section

- Bullet

Requirements / features

1. Number

Steps / sequence

Code / example / error message

### HTML Kya Hai?

Web page ki language.

* Colors

* Tables

* Diagrams

* Buttons

* Interactive controls

* Shareable links

### Sabse Important 5 Markdown Skills

1

Headings

AI ko hierarchy samajh aati hai.

2

Bullets

Order matter nahi karta.

3

Numbered Steps

Order important hai.

4

Code Blocks

AI ise data samjhta hai, instruction nahi.

5

Links

AI source page read kar sakta hai.

### Specification (Spec) Template

Out of Scope AI ko extra features banane se rokta hai.

### Kab HTML Maangna Chahiye?

| Situation                | Format     |
| ------------------------ | ---------- |
| Long report              | HTML       |
| Dashboard                | HTML       |
| Client presentation      | HTML       |
| Shareable page           | HTML       |
| Future AI context        | Markdown   |
| WhatsApp / LinkedIn post | Plain Text |

### HTML Request Example

Prompt:

Create a single HTML page for parents reading on a phone. Include a fee table, WhatsApp button, holiday banner, and one simple diagram.

### Publishing Options

* Claude Publish: instant public link

* GitHub Gist: file save + edit later

* GitHub Pages: permanent website

* Netlify: drag & drop hosting

### Important Terms

| Term              | Meaning                       |
| ----------------- | ----------------------------- |
| Markdown          | Structured plain text         |
| HTML              | Web page format               |
| Artifact / Canvas | AI ka live rendered page      |
| Spec              | Detailed instruction document |
| Hard Constraint   | Must-follow rule              |
| Out of Scope      | Jo nahi banana                |

### Golden Rule

Remember

Machine ke liye Markdown, insan ke liye HTML.

Yahi “Markdown In, HTML Out” ka core concept hai.
