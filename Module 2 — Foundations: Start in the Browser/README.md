### Topic: Foundations – Browser Mein Shuru Karein

#### Simple Summary

Foundations 6 basic courses hain jo har student ko AI use karne se pehle seekhne chahiye. In courses ke baad aap AI se apna kaam tez kar sakte hain ya AI Workers bana sakte hain.

#### Key Points

* Sab kuch browser mein seekha ja sakta hai.
* ChatGPT, Claude, ya Gemini use kar sakte hain.
* Koi software install karne ki zaroorat nahi.
* Yeh 6 courses AI ki strong foundation banate hain.

#### 6 Foundation Courses

1. **AI Asal Mein Kya Hai**

   * AI kaise kaam karta hai.

2. **AI Prompting**

   * AI se sahi sawal kaise poochna hai.

3. **Markdown In, HTML Out**

   * Basic document formats seekhna.

4. **Code Jo Aap Kabhi Nahin Likhte**

   * AI se code likhwana aur check karwana.

5. **Skills aur Connectors**

   * AI ko tasks sikhana aur apps se connect karna.

6. **AI ke Daur Mein Sochna**

   * Kab AI use karna hai aur kab nahi.

#### Important Terms

* **Foundation** = Basic skills
* **Prompting** = AI ko instructions dena
* **Skills** = AI ko sikhaye gaye tasks
* **Connectors** = AI ko apps se connect karne ka tareeqa

**Main Idea:** Pehle Foundations complete karo, phir General Agents aur AI Workers ki taraf jao. 📚
